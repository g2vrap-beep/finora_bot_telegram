# 💎 Moliya Bot — Инструкция по установке

## Что нужно (всё бесплатно):
- Аккаунт Telegram
- Аккаунт на [railway.app](https://railway.app) (бесплатно)
- Аккаунт на [github.com](https://github.com) (бесплатно)
- API-ключ Anthropic (claude.ai → настройки → API)

---

## Шаг 1 — Создать бота в Telegram

1. Открой Telegram, найди `@BotFather`
2. Напиши `/newbot`
3. Задай имя: `Moliya` (или любое)
4. Задай username: `moliya_моё_имя_bot`
5. Скопируй **токен** — выглядит как `7123456789:AAF...`

---

## Шаг 2 — Загрузить код на GitHub

1. Зайди на [github.com](https://github.com) → New repository
2. Назови: `moliya-bot` → Create
3. Загрузи все файлы из папки (bot.py, requirements.txt, Procfile, nixpacks.toml)

---

## Шаг 3 — Задеплоить на Railway

1. Зайди на [railway.app](https://railway.app) → Login with GitHub
2. Нажми **New Project** → **Deploy from GitHub repo**
3. Выбери `moliya-bot`
4. Перейди во вкладку **Variables** и добавь:

```
BOT_TOKEN    = токен_от_BotFather
CLAUDE_KEY   = sk-ant-...ключ_от_Anthropic
DB_PATH      = /data/finance.db
```

5. Перейди во вкладку **Settings** → **Add Volume**
   - Mount path: `/data`
   - (это для хранения базы данных)

6. Нажми **Deploy** — Railway сам установит всё!

---

## Готово! Открой своего бота в Telegram

Напиши `/start` — выбери язык и начни пользоваться.

---

## Команды бота:

| Команда | Описание |
|---------|----------|
| `/start` | Запуск и выбор языка |
| `/stats` | 📊 Статистика (баланс, доходы, расходы) |
| `/history` | 📋 Последние 10 записей |
| `/advice` | 🤖 AI-советы куда вложить деньги |
| `/rate` | 💱 Курс USD/EUR/RUB от ЦБ Узбекистана |
| `/clear` | 🗑 Очистить все данные |
| `/lang` | 🌐 Сменить язык (рус/узб) |

## Как добавлять транзакции:

**Текстом:**
- `Потратил 50000 сум на обед`
- `Получил зарплату 3 000 000 сум`
- `Купил кроссовки за 150 долларов`
- `Оплатил квартиру 800 000`

**Голосом:** просто запиши голосовое сообщение

**Фото:** отправь фото чека или скриншот оплаты

---

## Возможные проблемы:

**Бот не отвечает** → проверь переменные окружения в Railway (Variables)

**Голос не работает** → убедись что nixpacks.toml загружен (нужен ffmpeg)

**Ошибка Claude** → проверь правильность CLAUDE_KEY
