## 📝 Что ТЕБЕ нужно сделать (3 шага):

### ШАГ 1: Получи бесплатный GROQ_KEY

Это нужно чтобы бот понимал голос БЕСПЛАТНО.

1. Открой браузер
2. Иди на сайт: **https://console.groq.com/**
3. Нажми **Sign Up** (зарегистрируйся через Google/GitHub)
4. После входа нажми слева **API Keys**
5. Нажми **Create API Key**
6. Назови ключ: `finora-bot`
7. **СКОПИРУЙ КЛЮЧ** (он начинается с `gsk_...`)
8. Сохрани куда-нибудь, он пригодится в Шаге 2

---

### ШАГ 2: Добавь переменные в Railway

Это нужно чтобы бот знал как работать.

1. Открой твой проект на **https://railway.app**
2. Найди свой проект `finora_bot_telegram`
3. Нажми на него
4. Нажми вкладку **Variables** (слева)
5. Добавь **3 новые переменные** (нажимай **New Variable**):

```
Переменная 1:
Name: GROQ_KEY
Value: gsk_твой_ключ_который_скопировал_в_шаге_1

Переменная 2:
Name: DASHBOARD_URL
Value: https://твой-проект.up.railway.app
(скопируй свой URL проекта Railway - он есть в настройках)

Переменная 3:
Name: FLASK_SECRET_KEY
Value: abc123def456ghi789jkl012mno345pqr567stu890vwx
(любая случайная строка букв и цифр, минимум 32 символа)
```

6. **Сохрани** все переменные

## 🎉 ГОТОВО! Как проверить:

### Проверка 1: Голосовой онбординг

1. Открой своего бота в Telegram
2. Напиши: `/start`
3. Выбери язык кнопкой
4. Когда бот спросит "Как тебя зовут?" → **запиши голосовое сообщение**: _"Меня зовут Влад"_
5. Бот должен ответить: 🎤 _Меня зовут Влад_ ✅ Отлично!

✅ **РАБОТАЕТ!** Продолжай регистрацию голосом или текстом.

### Проверка 2: Веб-дашборд

1. В боте напиши: `/dashboard`
2. Бот отправит кнопку **🌐 Открыть Dashboard**
3. Нажми на кнопку
4. Откроется сайт → нажми **Login with Telegram**
5. Увидишь свою статистику с графиками! 📊

## 🆘 Если что-то не работает:

### Бот не понимает голос:
- Проверь что добавил `GROQ_KEY` в Railway Variables
- Подожди 2 минуты после добавления (Railway перезапускается)

### Дашборд не открывается:
- Проверь что добавил `DASHBOARD_URL` и `FLASK_SECRET_KEY`
- URL должен совпадать с адресом твоего Railway проекта

### Вообще ничего не работает:
- Открой Railway → твой проект → вкладка **Deployments**
- Нажми на последний деплой
- Смотри **Logs** - там будет ошибка
- Скинь мне скрин, разберемся!
