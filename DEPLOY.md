# 🚀 Деплой Finora Bot на Railway

## 📋 Что добавлено

### ✅ Голосовой онбординг
- Теперь пользователи могут проходить регистрацию полностью голосом
- Бот распознает имя, суммы доходов и финансовые цели через Whisper API
- Работает как на русском, так и на узбекском языке

### ✅ Веб-дашборд
- Персональная панель статистики для каждого пользователя
- Вход через Telegram (безопасно, без паролей)
- Графики доходов/расходов по месяцам
- Круговая диаграмма расходов по категориям
- Список всех транзакций
- Отслеживание прогресса к финансовой цели

## 🔧 Настройка Railway

### 1️⃣ Переменные окружения (Environment Variables)

Добавьте в Railway Settings → Variables:

```bash
# Обязательные переменные (уже есть)
BOT_TOKEN=ваш_токен_от_BotFather
OPENROUTER_KEY=ваш_ключ_openrouter
DATABASE_URL=автоматически_создается_Railway
DATABASE_PUBLIC_URL=автоматически_создается_Railway

# Для голосовых сообщений (хотя бы один)
GROQ_KEY=ваш_groq_api_key          # Бесплатный! Whisper-large-v3
# или
OPENAI_KEY=ваш_openai_api_key      # Платный ($0.006/мин)

# Новые переменные для дашборда
DASHBOARD_URL=https://ваш-проект.up.railway.app
FLASK_SECRET_KEY=любая_случайная_строка_32_символа

# Опциональные
OR_MODEL=anthropic/claude-sonnet-4-5  # AI модель для советов
```

### 2️⃣ Как получить GROQ_KEY (бесплатно!)

1. Зайди на https://console.groq.com/
2. Создай аккаунт
3. Перейди в API Keys → Create API Key
4. Скопируй ключ и добавь в Railway как `GROQ_KEY`

**Groq дает бесплатные лимиты:** 14,400 запросов в день!

### 3️⃣ Генерация FLASK_SECRET_KEY

В терминале запусти:
```bash
python -c "import secrets; print(secrets.token_hex(32))"
```

Скопируй результат и добавь в Railway как `FLASK_SECRET_KEY`

## 🎯 Деплой в Railway

### Автоматический деплой (рекомендуется)

1. **Push в GitHub:**
   ```bash
   cd finora_bot_telegram
   git add .
   git commit -m "✨ Добавлен голосовой онбординг и веб-дашборд"
   git push origin main
   ```

2. **Railway автоматически:**
   - Обнаружит изменения
   - Установит зависимости из requirements.txt
   - Запустит 2 процесса (bot + dashboard)
   - Выдаст публичный URL

3. **Обнови DASHBOARD_URL:**
   - Скопируй URL проекта (например: `https://finora-bot-production.up.railway.app`)
   - Добавь в Variables → `DASHBOARD_URL`
   - Railway автоматически перезапустится

## 🎮 Как пользоваться

### Для пользователей бота:

**В Telegram боте:**
```
/start      - Начать (можно пройти регистрацию голосом! 🎤)
/stats      - Посмотреть статистику
/dashboard  - Открыть веб-панель 🌐
/advice     - Получить AI-совет
/help       - Помощь
```

**Голосовое сообщение в онбординге:**
- Бот спросит: "Как тебя зовут?"
- Скажи голосом: "Меня зовут Влад" ✅
- Бот распознает и продолжит!

**Веб-дашборд:**
1. В боте: `/dashboard`
2. Нажми кнопку "🌐 Открыть Dashboard"
3. Войди через Telegram
4. Смотри свою статистику! 📊

## 🔐 Безопасность

✅ **Telegram Login Widget:**
- Проверка подписи от Telegram
- Данные действительны 24 часа
- Невозможно подделать без BOT_TOKEN

✅ **Конфиденциальность:**
- Каждый пользователь видит только свои данные
- Сессии хранятся в Flask session (HTTP-only cookies)

✅ **База данных:**
- PostgreSQL с SSL
- Данные хранятся в Railway (защищенная инфраструктура)

## 🐛 Решение проблем

### Бот не распознает голос
**Проблема:** Нет GROQ_KEY и OPENAI_KEY  
**Решение:** Добавь хотя бы один ключ в Railway Variables

### Дашборд не открывается
**Проблема:** Не настроен DASHBOARD_URL  
**Решение:** Скопируй URL проекта Railway и добавь в Variables

### Ошибка "BOT_TOKEN not set"
**Проблема:** Не заданы переменные окружения  
**Решение:** Проверь Railway Settings → Variables

### Telegram Login не работает
**Проблема:** Неправильный домен в коде  
**Решение:** Убедись что DASHBOARD_URL совпадает с реальным доменом Railway
